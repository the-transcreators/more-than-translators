WEBSITE LOCALIZATION AS EASY AS IT GETS

With website work, we normally just deal with the text. We are not web designers. But some customers want more. They are looking for a no-frills way to get a simple website designed and built.

In response to that, we have developed a template to help people quickly get a low-cost English version of their website. It is particularly attractive for small businesses that don't have a huge marketing budget, but who do want to do the following:

  • Get a straightforward, static website that is clean, modern and clear
  • Spend as little on web design as possible
  • Be able to easily self-edit the site for free in the future

We always use the same familiar, simple, template (that's one reason why the cost is so low)

Not only are we not website designers, we do not want to become website designers either. Nor do most of our customers. So the one-and-only website template we use is extremely simple. That's a good thing for us, and for our customers. We use this template the same way every time. We never spend hours trying to change something. We did spent a lot of time at the beginning on developing a template that does not need to be changed in order to produce a website that is clean, modern and clear.

What people pay for is the cost of transcreating their text. The vast majority of our effort goes into the words. The words are one of the two big things that will make the message powerful. The other big thing is images. Customers provide us with the images they want to include, or add images themselves later.

We transcreate copy directly into a simple, free and easy-to-use website builder (that's another reason why the cost is so low)

Working directly in a simple website builder hardly takes any longer than working in Word. So the website-building work only adds a tiny extra cost. We use a free, open source, offline website builder that is not tied to any web hosting service. The user interface is intuitive and does not require any coding.

For anyone who likes the philosophy behind our approach and wants the benefits, then the English version of their website must be created using the one-and-only template we use.

This solution is only suitable if it is okay for the English website to look different from the main website.

We hand over the project to customers and they can easily use the free website builder to make changes to their site

The project is offline and is stored on the owner's computer. The editor is open source, installable, offline software. Customers have complete control of their project.

If they want to make changes at any time, they can easily do so themselves for free. If it is weeks or months since the last time changes were made, there is still no need to worry about how to work with the editor. It is so simple that it can be learned or re-learned in minutes.

Some people like the new English version of their website so much that they decide to adopt the same website builder and template for their main site

And why not? It will look great. And the editor is easy for normal humans to use. And it's free. And it is NOT bundled with a payment plan that costs money month after month.

Simplicity and independence. Wonderful.
